# 竞价分析策略 · 自主学习框架

> 侦察兵 v3.0 增强方向。5 维度竞价特征 → 模式分类 → 日内验证 → Bayesian 权重迭代。

## 数据源

东方财富 API (`push2.eastmoney.com/api/qt/stock/get`)，09:15-09:25 每 3 秒更新：

| 字段 | 含义 | 策略价值 |
|------|------|:--:|
| `f43` | 虚拟匹配价 | 价格轨迹 — 竞价方向 |
| `f19` | 虚拟匹配量 | 量能加速度 — 抢筹/抛压 |
| `f20` | 虚拟成交额 | 资金力度 |
| `f177/f178` | 涨停/跌停价 | 封板预期边界 |

## 五大竞价策略

| # | 策略 | 公式 | 信号 |
|:--:|------|------|------|
| 1 | **价格轨迹斜率** | `slope = (P₉₂₅ − P₉₁₅₋₁₈) / Δt` | 正=抢筹，W型=分歧转一致(最强) |
| 2 | **量能加速度** | `accel = V₉₂₅₋尾 / V₂₀₋₁₅头` | >1.5=尾段爆量涌入 |
| 3 | **委托不平衡** | `imbalance = (bid−ask)/total` | >0.3买方主导 |
| 4 | **开盘溢价率** | `premium = (P_open−P_close)/P_close` | 2-5%最理想，>7%高开低走风险 |
| 5 | **板块联动** | `corr(stock, sector_avg)` | 独立上攻 > 跟随 > 逆板块 |

## 自主学习循环

```
09:15-25  竞价采集器 → 200帧/只 × Top40只 → SQLite 存储
09:25     模式分类器 → 5维特征提取 → 信号输出 → 侦察兵增强
15:00     日内验证 → 实际涨跌 → 标签写入(命中/未命中)
23:00     权重优化器 → Bayesian Update → α/(α+β) → 新权重
```

每个策略维度维护独立的 Beta 分布 `(α, β)`：
- 命中：α += 1
- 未命中：β += 1
- 后验权重：α / (α + β)
