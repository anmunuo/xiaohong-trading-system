# 场外看涨分成 · 盈亏场景

## 参数

| 参数 | 值 |
|:--|:--|
| 保证金 | ¥100,000 |
| 期权费 | ¥8,000 |
| 名义本金 | ¥1,000,000 |
| 强平线 | 保证金 ≤ 20% |

---

## 场景1：大涨 +15%

股价从 ¥12.00 → ¥13.80，涨幅 +15%

```
gross_profit    = 15% × 1,000,000 = ¥150,000
total_profit    = ¥150,000 - ¥8,000 = ¥142,000
个人分成 70%     = ¥142,000 × 0.70 = ¥99,400
券商分成 30%     = ¥142,000 × 0.30 = ¥42,600
保证金健康度     = 100%（无损）
实际回报率       = ¥99,400 / ¥108,000 = 92.0%
```

---

## 场景2：下跌 -5%

股价从 ¥12.00 → ¥11.40，跌幅 -5%

```
毛亏损          = 5% × 1,000,000 = ¥50,000
保证金消耗       = ¥50,000
保证金剩余       = ¥100,000 - ¥50,000 = ¥50,000
保证金健康度     = 50%（🟡 warning）
个人净亏         = ¥50,000 + ¥8,000 = ¥-58,000
```

---

## 场景3：暴跌 -9% → 强平

股价从 ¥12.00 → ¥10.92，跌幅 -9%

```
毛亏损          = 9% × 1,000,000 = ¥90,000
保证金消耗       = ¥90,000
保证金剩余       = ¥100,000 - ¥90,000 = ¥10,000
保证金健康度     = 10%（🔴 liquidate）
个人净亏         = ¥90,000 + ¥8,000 = ¥-98,000
→ 强制平仓：保证金基本归零
```

---

## 场景4：微涨 +0.5%（不够覆盖期权费）

股价从 ¥12.00 → ¥12.06，涨幅 +0.5%

```
gross_profit    = 0.5% × 1,000,000 = ¥5,000
total_profit    = ¥5,000 - ¥8,000 = ¥-3,000
→ total_profit < 0 → 不触发分成
个人净亏         = ¥-3,000（期权费未回本）
保证金健康度     = 100%（未动用）
```

**结论**：需要股价涨 ≥ 0.8% 才能覆盖期权费开始真正盈利。
