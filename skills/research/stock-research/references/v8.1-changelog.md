# v8.1 更新日志

## 2026-06-02 — v8.1

### 推荐引擎修复

| 问题 | 根因 | 修复 |
|------|------|------|
| 超时 (cron error) | `_prefetch_indicators` 串行 `subprocess.run` ~100-300s | `ThreadPoolExecutor(max_workers=8)` 并行 → 10-25s |
| 停牌股漏入 | 停牌检测仅在 enrichment 层 | 新增四重过滤：`_get_suspended_codes()` — KB洞察+行情双路检测 |
| 市值过滤失效 | `get_stock_realtime`(Sina) 无 `market_cap`，全部返回 None | tushare `daily_basic` 逐只查询 `total_mv`(万元→亿元) |
| 大市值漏网 | 招商银行 9634亿 入池 | 修复后 `excluded.large_cap=14` |
| 所有评分相同 | 五因子退化 | 注入 macro 数据(北向/板块/外盘/行业新闻/宏观政策) |
