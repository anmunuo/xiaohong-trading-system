# v8.10 技能变更日志 (2026-06-04)

## 本次修复

### P0: 22/26 Cron 脚本行号污染

- **症状**: cron job 报 `script failed` / `未找到命令`
- **根因**: `cron_*.sh` 文件内容被写入了带行号前缀（`     1|#!/bin/bash`），bash 无法解析 shebang
- **影响**: 侦察兵(开盘+盘中4次)、弹药库、竞价采集器、文工团复盘、市场快照、研究员研学等
- **修复**: `sed` 批量去行号 + `bash -n` 语法验证
- **防范**: `system_health_check.py` 维度12「📜 Cron脚本」每日3次自动扫描 shebang 开头

### P0: 北向资金实时通道永久关闭

- **症状**: AKShare 北向 `成交净买额` 恒为 0，管线返回 "0.0亿净流入" 多日不变
- **根因**: 2024年5月起交易所不再实时披露北向资金，但 `get_north_flow()` 择优逻辑 `if 日期新鲜 OR net_flow!=0` 永远走 AKShare
- **修复**: `if net_flow != 0` 才信 AKShare，否则回退 tushare T-1 日终数据
- **额外修复**: tushare 回退窗口 7→30 天；新增 `_quality: T-N` 标记数据滞后

### P1: review.py 导入失败

- **症状**: `ImportError: cannot import name '_em_api_get' from 'data_pipeline'`
- **根因**: v8.5 data_pipeline 拆分子模块后 `_em_api_get` 未从 `__init__.py` 重导出
- **修复**: `from data_pipeline._core import _em_api_get`
