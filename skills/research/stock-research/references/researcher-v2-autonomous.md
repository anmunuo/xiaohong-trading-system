# 研究员议会 v2.0 · 自主学习者

> 2026-06-04 · 从 v1.0 升级

## 架构变更

```
v1.0:  context → analyze() → 模板报告 → 丢弃
v2.0:  context → analyze() + verify_past → 报告
            ↓                    ↓
      生成新假设(Hypothesis)    更新已有假设信念
            ↓                    ↓
       researcher_state.json (持久化)
            ↓
  次日: 自动验证 → Bayesian α/β → 确认/驳回
```

## 新增能力

| 能力 | 机制 | 文件 |
|:--|:--|:--|
| 假设提出 | `_make_hypothesis()` 在分析过程中自动生成 | `researchers.py` |
| Bayesian信念 | α+1(支持)/β+1(反对) → 置信度自动更新 | `Hypothesis.update_belief()` |
| 跨日记忆 | `researcher_state.json` 持久化 | `ResearcherState` |
| 收盘验证 | `verify_past_hypotheses()` 自动对比 | 各研究员 `_verify_one()` |
| 自动归档 | 置信度≥80%→confirmed, ≤20%→rejected | `Hypothesis` status |
| 数据缺口追踪 | `state.gaps` 持续累积 | `ResearcherState.add_gap()` |

## 资金面研究员

新增第6位研究员，专注全市场资金流向与量价关系：

- 北向资金（含T-N滞后标记）
- 主力/散户资金背离检测
- 板块资金流入持续性
- 融资融券杠杆情绪
- 量价关系（放量上涨/放量下跌/异常放量）
- 资金风格判断（机构vs游资）

## 关键教训

研究员不能只是统计脚本——数公告、查时效、打分的分析没有学习价值。真正的自主学习需要：假设→验证→信念更新→知识积累的闭环。
