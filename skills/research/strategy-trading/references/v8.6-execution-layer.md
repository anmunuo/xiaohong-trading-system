| 模块 | 文件 | 入口 |

| TWAP/VWAP算法执行 | `algo_executor.py` | `--twap CODE QTY` / `--vwap CODE QTY` |

## TWAP 执行

基于时间等分拆单，减少单笔冲击。

```bash
python3 algo_executor.py --twap 600519 10000 --slices 10
# 输出: 10个时间切片，每片1000股
# 预计冲击: 5.9 bps → TWAP可减至 4.1 bps
```

## VWAP 执行

基于历史分时成交量分布加权拆单。

```bash
python3 algo_executor.py --vwap 600519 10000
# 输出: 8个时间切片（基于实际量分布）
# 尾盘30分钟配18%量，开盘配12%
```

## 券商网关

Paper/LIVE 统一接口：

```python
from broker_gateway import BrokerGateway, BrokerType
gw = BrokerGateway(BrokerType.PAPER)  # 或 BrokerType.XTQUANT
gw.connect()
acc = gw.get_account()  # → AccountInfo
order = gw.submit_order("600519", OrderSide.BUY, 1800, 100)
```

## 冲击成本估算

Almgren-Chriss 简化模型：
- σ = 20日波动率
- Q/V = 参与率
- 冲击 ≈ σ × (Q/V)^0.5 × 10000 bps
- TWAP 约减 30%, VWAP 约减 45%
