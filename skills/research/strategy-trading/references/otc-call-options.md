## 产品参数

```
┌─────────────────────────────────────────┐
│  场外看涨分成 (OTC Call + P&L Split)      │
├─────────────────────────────────────────┤
│  保证金    ¥100,000     风险资金            │
│  期权费    ¥8,000       固定通道成本         │
│  名义本金  ¥1,000,000   对应股票市值敞口      │
│  ─────────────────────                   │
│  上涨 → 个人 70% : 券商 30%               │
│  下跌 → 个人 100%（保证金扣除）             │
│  强平 → 保证金 ≤ 20%（亏 ≥ ¥80,000）      │

└─────────────────────────────────────────┘
```

## 盈亏计算逻辑

### 价格上涨

```
gross_pnl = (current_price - entry_price) / entry_price × notional
after_fee = gross_pnl - option_fee
personal = after_fee × 0.70  (如果 after_fee > 0)
broker   = after_fee × 0.30
```

- 如果 `gross_pnl < option_fee`（涨幅不够覆盖期权费）→ 个人承担全部，broker 为 0
- 保证金不受影响（保持 ¥100,000）

### 价格下跌

```
loss = abs(gross_pnl)
margin_consumed = min(loss, margin)
margin_remaining = margin - margin_consumed
margin_health = margin_remaining / margin
personal_pnl = -margin_consumed - option_fee
```

- 保证金健康度 ≤ 20%（亏损 ≥ ¥80,000）→ 强制平仓
- 亏损先从保证金扣，期权费是固定损失

## 保证金健康度分级

| 健康度 | 级别 | 状态 | 动作 |
|:--|:--|:--|:--|
| 100% ~ 61% | 0 | 🟢 ok | 正常 |
| 60% ~ 41% | 1 | 🟡 warning | 关注 |
| 40% ~ 21% | 2 | 🟠 danger | 准备补保/减仓 |
| ≤ 20% | 3 | 🔴 liquidate | 强制平仓 |

## 数据类型说明

场外期权 ≠ 场内 ETF 期权：

| 维度 | 场外 OTC | 场内 ETF 期权 |
|:--|:--|:--|
| 数据源 | 券商报价（手工录入） | AKShare `option_sse_*` |
| 行情 | 无连续公开行情 | 实时竞价 |
| 期权链 | 无公开 API | AKShare 全量 |
| Greeks | 自算（BS 模型） | AKShare `option_sse_greeks_sina` |
| IV | 从历史波动率估算 | AKShare QVIX 指数 |
