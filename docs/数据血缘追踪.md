# 小红交易系统 · 数据血缘追踪

## 分层数据流全景

```
外部API                         Bronze 层                    Silver 层                Gold 层
══════════                      ═══════════                 ════════════            ════════════
Sina q=批量行情 ──→  daily_kline/YYYY/MM/DD/sina.json.gz ──→ stock_daily/all.json.gz ──→ factor_panel/v3.parquet
                                                        │                           │
BaoStock K线 ────→                          (通过钩子)   │  清洗规则:                 │  因子计算 (26维):
                                                        │  · 前复权统一              │  · 动量 mom_5d/20d/60d/alpha
东财 push2 ──────→  fund_flow/YYYY/MM/DD/eastmoney.json.gz  · 停牌forward-fill        │  · 波动 atr_14/vol_20d/downside_vol
                                                        │  · 异常价格标记            │  · 资金 net_flow/turnover/vol_ratio
tushare ─────────→  fundamentals (via hook)              │  · 质量flags              │  · 筹码 turnover_zscore/vol_ratio_trend/amplitude
                                                        │                           │  · 估值 pe_ttm/pb/total_mv_log
AKShare ─────────→  market_index/.../akshare.json.gz     │  每行=代码×日期            │  · 质量 is_suspended/is_st/n_quality_flags
                 └→ events/.../akshare.json.gz          │  不可变                    │  · 滚动 close_zscore/volume_zscore/pe_percentile
                                                        │                           │    pb_percentile/ma5_deviation/ma20_deviation
Sina分时 ────────→  intraday/YYYY/MM/DD/sina.json.gz     │  stock_master: 5524只 ←──┤  · 数据源: Silver + Bronze启动期
                                                        │                           │  · 每次重算一致 ✅
                                                        │                           │
                                                        │                           ├──→ ml_datasets/train_YYYYMMDD_v3.npz
                                                        │                           │    (X=因子面板, y=次日涨跌)
                                                        │                           │
                                                        │                           └──→ daily_pool/YYYY/MM/DD.json
                                                        │                                (推荐池历史归档)
```

## 三层架构数据流

```
Bronze (不可变)          Silver (清洗)            Gold (特征)
═══════════════          ════════════            ═══════════════
bronze_ingest.py    →    silver_pipeline.py  →    gold_pipeline.py
  (15:40)                   (15:45)                 (15:50)

数据源:                   数据源:                  数据源:
  · 外部API               · Bronze only            · Silver (主)
  · 6类数据                · 5524只全A主表           · Bronze (启动期补充)
                            · 质量标记               · 26维因子计算
                            · 不可变清洗              · ML数据集构建
                                                     · daily_pool归档
```

## 可复现性验证

```bash
# 删除 Silver 和 Gold，从 Bronze 重建
rm -rf data/silver/stock_daily/2026/06/03/
rm -rf data/gold/factor_panel/2026/06/03/

# 重建 Silver
python3 scripts/silver_pipeline.py --date 2026-06-03

# 重建 Gold
python3 scripts/gold_pipeline.py --date 2026-06-03

# 验证 Silver
python3 scripts/silver_verifier.py --date 2026-06-03

# 验证 Gold
python3 scripts/gold_verifier.py --date 2026-06-03

# 验证 Gold 可复现性
python3 scripts/gold_pipeline.py --verify --date 2026-06-03
# → ✅ 可复现: 100只股票完全一致
```

## 每日数据流水线时间线

```
15:00 收盘
15:05 ⏱️  分时K线冻结 → Bronze intraday/
15:30 🛡️  弹药库风控
15:35 📊  股票跟踪器
15:40 🗄️  Bronze全量采集 → daily_kline + fund_flow + events
15:45 🥈  Silver ETL      → Bronze→清洗→统一格式 (5524只)
15:50 🏆  Gold ETL        → Silver+Bronze→因子面板+ML+Pool归档
17:00 📊  因子IC评估      → 读 Gold factor_panel
17:10 🤖  ML增量训练      → 读 Gold ml_datasets/*.npz
17:20 📈  组合回测        → 读 Gold daily_pool 历史
17:30 🧬  进化引擎
```
